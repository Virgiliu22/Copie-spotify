🎧 Spotify Clone

Un mic proiect făcut de noi în care am încercat să recreăm un player muzical asemănător cu Spotify, folosind HTML, CSS și JavaScript.

Nu este o copie perfectă, dar ideea a fost să învățăm cum funcționează o aplicație de tip music player și să facem ceva cât mai apropiat de realitate.

💡 Despre proiect

Proiectul reprezintă un site web unde poți reda muzică, schimba melodii și vedea un fel de playlist.

Am vrut să facem ceva interactiv, nu doar un site simplu, de asta am adăugat funcții cu JavaScript.

🔥 Ce am reușit să facem
🎧 Player muzical
play / pause

bară de progres
